### Technology Stack

- Vue3
- Sass - Component styling
- Jest - componnet unit tests
- Story Book

### Prerequisites

- Yarn
- [Visual Studio Code](https://code.visualstudio.com/) with [Vetur extension](https://github.com/vuejs/vetur)

### Getting Started

#### Install dependancies

```
yarn install
```

#### Start Storybook

```
yarn dev
```

#### Build Storybook

```
yarn workspace oxd-storybook build
```

#### Unit Testing

```
yarn test:unit
```

#### Linting

```
yarn lint
```
